# EcoCity - Sistema de Gestão de Resíduos Inteligente

O **EcoCity** é um projeto em desenvolvimento pensado para otimizar a logística reversa e o descarte consciente de resíduos urbanos. O sistema gamifica o descarte de materiais (recicláveis, orgânicos e tóxicos), convertendo o peso dos itens em "Pontos Verdes" para o cidadão.

### 🚀 Tecnologias Utilizadas
- **Java 17+**
- **Orientação a Objetos (OO)**
- **VS Code**

### 🧠 Conceitos Aplicados
Este projeto faz parte do meu plano de estudos intensivo em Backend Java, onde aplico:
- **Encapsulamento:** Proteção de dados sensíveis e regras de negócio através de modificadores de acesso (`private`) e métodos acessores.
- **Lógica de Negócio Customizada:** Cálculo dinâmico de pontuação baseado no tipo de material e risco ambiental.

### 🛠️ Funcionalidades Atuais (MVP v1.0)
- [x] Cadastro de materiais com atributos de peso e toxicidade.
- [x] Motor de cálculo de pontos com bônus para materiais Orgânicos e Tóxicos.
- [x] Interface via Console para visualização de resultados.
